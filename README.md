### 🎯 Миссия "Народного Футбола"
Мы создаем глобальный социальный лифт. Наша цель — найти таланты в обычных городских коробках и, используя высокие технологии, вывести их на профессиональные арены стран БРИКС+.

### 🚀 Ключевые этапы пути:
1. **Отбор (Smart-Field):** Цифровая оценка талантов в 11 городах Крыма.
2. **Городское Дерби:** Битва двух составов за право представлять свой город.
3. **Региональный этап:** Игры на больших профессиональных стадионах.
4. **Мировой финал:** Гранд-матч в Ялте на стадионе «Авангард» и выход на уровень БРИКС+.
