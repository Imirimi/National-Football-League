### 🎯 Концепция проекта
«Народный футбол» — это социальный лифт для самородков. Мы создаем экосистему, где таланты из обычных дворовых "коробок" выходят на профессиональные стадионы и международную арену БРИКС+. 

*Проект поддерживает интеграцию с государственной системой «Навигатор» для верификации достижений юных участников.*

### 🚀 Этапы развития:
1. **Городской отбор:** Битва в "коробках", создание двух команд города и "Первого состава".
2. **Межгородской кубок:** Матчи на больших профессиональных стадионах регионов.
3. **Гранд-Финал в Ялте:** Формирование Золотой Сборной России для мирового вызова.
