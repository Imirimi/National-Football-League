## Глобальная концепция
«Народный Футбол» — это не просто лига, это социальный лифт. Мы создаем систему, где каждый талантливый ребенок (мальчик или девочка) может попасть из дворовой коробки в большой спорт благодаря цифровому следу.

### Наши цели:
* **Цифровизация:** Оцифровка 100% игроков через систему объективной оценки.
* **Масштаб:** Старт в Крыму (14 городов) с последующим выходом на федеральный уровень России и страны БРИКС+.
* **Прозрачность:** Прямой доступ скаутов к базе данных талантов без посредников.
