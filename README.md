## О проекте
«Народный Футбол» — это революция в поиске спортивных талантов. Мы объединяем дворовую искренность и космические технологии, создавая социальный лифт для детей и молодежи стран БРИКС+.

### Миссия
Дать шанс каждому игроку, независимо от места проживания, быть замеченным мировыми скаутами через систему объективного цифрового анализа.
